Cybrige Certified Android Pentester (CCAND)

Live Instructor-Led Training for Android Mobile Security Testing

Course Overview

The Cybrige Certified Android Pentester (CCAND) program is designed to equip you with hands-on skills in Android mobile security testing and vulnerability assessment. This comprehensive course covers mobile threat modeling, static and dynamic analysis, reverse engineering, secure coding practices, and real-world exploitation techniques. Learn to identify and exploit Android security flaws through practical, instructor-led sessions that simulate real-world mobile application security scenarios.

Android Security Course Content

Module 01: Introduction to Android Security

Understand Android architecture, security model, components, and common attack vectors in mobile applications.

Module 02: Android Application Structure

Learn Android app components, manifest files, permissions, intents, and how applications are structured.

Module 03: Mobile Threat Modeling

Understand threat modeling specific to mobile applications and identify potential security risks.

Module 04: Static Analysis Fundamentals

Learn techniques for analyzing Android apps without execution, including decompilation and code review.

Module 05: Dynamic Analysis Setup

Set up environments for dynamic analysis including emulators, physical devices, and testing tools.

Module 06: APK Analysis and Decompilation

Master techniques to extract, decompile, and analyze APK files to understand application logic.

Module 07: Reverse Engineering Android Apps

Learn reverse engineering techniques to understand application behavior and discover vulnerabilities.

Module 08: Runtime Manipulation

Use tools like Frida and Xposed to manipulate app behavior at runtime and bypass security controls.

Module 09: Android Permissions and Security

Analyze permission models, identify overprivileged apps, and exploit permission-related vulnerabilities.

Module 10: Insecure Data Storage

Discover and exploit insecure data storage vulnerabilities including SQLite, SharedPreferences, and file storage.

Module 11: Insecure Communication

Test for insecure network communication, SSL/TLS misconfigurations, and certificate pinning bypasses.

Module 12: Code Injection and Obfuscation

Understand code injection attacks, app obfuscation techniques, and methods to bypass protection mechanisms.

Module 13: Root Detection and Bypass

Learn how apps detect root access and techniques to bypass root detection mechanisms.

Module 14: Secure Coding Practices

Understand secure coding practices for Android development and how to identify insecure implementations.

Module 15: Mobile Pentesting Methodology

Learn comprehensive methodology for conducting security assessments of Android applications.

Module 16: Reporting Mobile Security Findings

Learn how to document and report Android security vulnerabilities effectively for stakeholders.
